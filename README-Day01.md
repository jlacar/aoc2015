# Day 1: Not Quite Lisp

A nice [warm-up puzzle](https://adventofcode.com/2015/day/1) to kick things off. This puzzle involves parsing a series of parentheses. Starting from the ground floor (0), each parenthesis in the input tells him to go up if it's an open parenthesis, `(`, or down if it's a close parenthesis, `)`, one floor.
